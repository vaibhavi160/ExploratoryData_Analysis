# ExploratoryData_Analysis
## Dataset
- Source: GeeksforGeeks
- Features include:
  - Fixed acidity
  - Volatile acidity
  - Citric acid
  - Residual sugar
  - Chlorides
  - Free sulfur dioxide
  - Total sulfur dioxide
  - Density
  - pH
  - Sulphates
  - Alcohol
  - Quality (score between 0â€“10)
  - Id

## Objectives
- Understand distribution of wine quality scores
- Explore correlations between chemical properties and wine quality
- Identify key features that influence wine quality
- Visualize relationships using plots and statistical summaries

## Tools & Libraries
- Python (Pandas, NumPy, Matplotlib, Seaborn, Plotly)
- Jupyter Notebook for interactive analysis
## Visualizations
- Histograms of wine quality distribution
- Heatmap of feature correlations
- Boxplots comparing chemical properties across quality levels
- Pairplots for multivariate relationships
